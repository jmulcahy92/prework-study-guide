Prework Study Guide

HTML

  • HTML stands for Hypertext Markup Language.
  • HTML is used to build a framework for webpages using elements such as text, image, header, and list elements.
  • The head element contains information about the webpage, sometimes called "metadata," and is not visible to users.
  • The body element represents the visible content shown to the user.
  • The available heading elements from largest to smallest are "h1", "h2", "h3", "h4", "h5", and "h6". They should be used in order and "h1" should only be used once per page.
  • The "p" element represents a paragraph or block of text.
  • The "ul", "ol", and "li" elements represent unordered lists, ordered lists, and list items respectively.
  • The "img" element contains information about images that displayed on the webpage.
  • The "br" element creates a line break between two blocks of content.
  • The "a" element creates links to the same webpage or other webpages. "A" in this context stands for "anchor."

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS is used to define the style of a webpage.
  • The three ways to style a webpage using CSS are inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS means styling elements directly in the HTML file.
  • Internal style sheets set rules for the entire page by adding code in the head section of the HTML file.
  • External style sheets set rules for the entire page by writing code in a separate file and linking to that file in the head section of the HTML file.
  • Using external style sheets is preferred in most cases because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • CSS rules are formatted thusly: "selector {declaration1; declaration2; etc}"
  • Declarations are formatted thusly: "property: value"
  • A selector defines the element or attributes to which the declarations apply.
  • Declarations are the rules our HTML elements will follow.
  • Examples of declarations are "color: blue" and "text-align: justify"
  • The * symbol is used as a selector for all elements visible on the page.
  • If one wishes to add declarations to some elements but not all of them, they can do so by designating a class attribute as the selector and then assigning that class attribute to the appropriate elements in the HTML file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds modifications in the current working branch to the staging area; the "-A" flag indicates we want to add all changes
  • git commit -m "commit notes": creates a snapshot of the currently staged changes; the "-m" flag associates a message with our commit, which we use to scuuinctly describe the changes implemented
  • git pull: receives a branch's modifications into the local environment; the "origin" keyword indicates the source of the pull will be in the GitHub repo; the "main" keyword indicates the branch
  • git push: pushes locally-made changes to the remote GitHub branch
